關於我

我是剛升上 資管系大三 的影迷愛好者，日常學習 資料分析 / SQL / Python， 平常喜歡觀看各種類型的電影，像是劇情、科幻、浪漫和驚悚都有。

正嘗試將對電影的熱情結合資管技能：整理觀影資料、抓取 TMDb / IMDb API、做推薦與視覺化，並偶爾剪一支預告重剪（re-cut）。 目標是打造 好維護、可擴充的電影資料應用， 也持續練習 影評寫作。
